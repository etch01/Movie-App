# Movie-App

# Installation:

First clone the app then do:

````
```
npm install
```
````

````
```
cd ios && pod install
```
```` 
  
If you are using Apple Silicon (M1) and pod install failed use this 

````
```
sudo arch -x86_64 gem install ffi
```
````

````
```
arch -x86_64 pod install
```
````

# Run :

````
```
npx react-native run-ios
```
````
